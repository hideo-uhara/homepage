GPSLoggerS Trackのプライバシーポリシー

  1. 表示したGPSログの履歴を表示するためにアプリ内のみにGPSログのファイル名を保存します。
  2. スタンドアロンのアプリなので、外部サーバーとの情報連携はありません。

GPSLoggerS Track Privacy Policy

  1. Save the GPS log file name only in the app to display the history of the displayed GPS log.
  2. Since it is a standalone application, there is no information linkage with external servers.
